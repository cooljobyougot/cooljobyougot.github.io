01 · идея

Главная мысль

Пассажиру не нужна задача. Он хочет, чтобы о нём уже позаботились.

До

Аэропорт, багаж, спешка, шум, усталость.

Решение

Maybach подъезжает вовремя. Водитель встречает и берёт заботу на себя.

После

Тишина, мягкий салон, ночной город, ощущение контроля.

02 · сценарий 55–60

Посекундная раскадровка

0–8 c
Героиня выходит из аэропорта. Багаж, шум, усталость, суета.

8–15 c
Появление Maybach

Подъезжает автомобиль. Фары, отражения, крупные кадры деталей.

15–23 c
Сервис

Водитель открывает дверь, помогает с багажом, действует точно, уверенно.

23–32 c
Салон

Дверь закрывается. Шум исчезает. Остаются тишина, приглушенный свет и комфорт. Кадры из салона, смена настроения героини.

32–45 c
Ночной город

Maybach едет по городу. Огни, диски, плавный ритм, общие планы, съёмка с дрона.

45–60 c
Состояние

Героиня расслабляется. Поездка становится частью отдыха.
