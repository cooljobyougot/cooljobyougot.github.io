# КАКАЯУВАСИНТЕРЕСНАЯРАБОТА.РФ — static site

Статический сайт креативного агентства **КАКАЯУВАСИНТЕРЕСНАЯРАБОТА.РФ**.

Сайт опубликован через GitHub Pages и используется как публичная витрина агентства, продуктовых страниц, коммерческих предложений, проектных материалов и контактной точки для заявок.

## Главная страница

* `/index.html` — главная страница агентства.
* `/` — основная публичная точка входа.

## Основная структура

Все важные страницы, кроме главной, лежат в папке:

* `/products/`

Это важно для SEO, sitemap, Яндекс Вебмастера, внутренних ссылок и AI-карт сайта.

## Главные коммерческие страницы

Ключевые посадочные страницы для органического поиска:

* `/products/event-visual-production.html` — видеосъёмка мероприятий, визуальное сопровождение форумов, конференций, корпоративных, городских и деловых событий.
* `/products/sport-extreme.html` — видеосъёмка спорта, экстрима, гонок, забегов, соревнований и активных событий.
* `/products/fpv-attraction-wow-offer.html` — FPV-аттракцион и wow-формат для мероприятий, площадок, брендов и городских пространств.
* `/products/concert-fpv-offer.html` — FPV-видеосъёмка концертов, сцен, артистов и музыкальных событий.
* `/products/air-balloon-fpv-video-offer.html` — FPV-видео полёта на воздушном шаре, съёмка аэростатов, романтических программ, подарков, love story и групповых полётов.
* `/products/beauty.html` — коммерческое предложение для beauty-индустрии: видеовизитки мастеров, салонов, студий, академий, экспертов и косметических брендов.
* `/products/show.html` — коммерческое предложение для шоу-индустрии: видеосъёмка шоу, трюковых программ, авто- и мото-событий, фестивалей и экшн-форматов.
* `/products/kids-sport.html` — коммерческое предложение для детских спортивных мероприятий: ролик события для организатора и персональные клипы участников для родителей.

## Технические заметки

Сайт статический и опубликован через GitHub Pages.

Основные папки:

* `/products/` — продуктовые, коммерческие и проектные страницы.
* `/assets/` — изображения, видео, стили и другие ассеты.
* `/meta/` — мета-файлы, иконки, OG-изображения.
* `/legal/` — юридические страницы.
* `/green/` — проектные материалы, если используются.
* `/Сценарий/` — сценарные материалы, если используются.

Сайт может использовать:

* встроенные стили внутри HTML-страниц;
* локальные изображения и видео;
* JavaScript для интерфейса, темы, навигации и возврата на предыдущую страницу;
* Яндекс Метрику на ключевых коммерческих страницах.

## Главный принцип

Реальная коммерческая структура сайта находится в конкретных продуктовых страницах и КП внутри папки `/products/`.

Новые универсальные SEO-страницы имеют приоритет над старыми клиентскими и проектными версиями, если пользовательский запрос не содержит название конкретного проекта или клиента.
